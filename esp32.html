ESP32-WROOM-32E Pinout

Power & Boot Notes

  • 3V3: regulated 3.3 V output / VCC rail.
  • EN: chip enable (active high) — tie high to run, pull low to reset.
  • GND: common ground.
  • Boot straps: GPIO0 / GPIO2 / GPIO5 / GPIO12 can affect boot mode depending on board.

I²C (typical)

SCL: GPIO22 SDA: GPIO21

You can move these if a library supports custom pins.

Analog

  • ADC1: GPIO36, 39, 34, 35, 32, 33
  • ADC2: GPIO4, 12–15, 25–27
  • DAC: GPIO25 (DAC1), GPIO26 (DAC2)

Serial & SPI

  • UART0: TX0 GPIO1, RX0 GPIO3 (USB-serial on most dev boards)
  • UART2: TX2 GPIO17, RX2 GPIO16
  • VSPI: SCK GPIO18, MISO GPIO19, MOSI GPIO23, CS GPIO5
  • HSPI: SCK GPIO14, MISO GPIO12, MOSI GPIO13 (alt bus)

PWM (LEDC)

Most GPIOs support PWM via LEDC except the “input-only” pins (GPIO34–39).
